Pviz

Pviz is an R package for peptide visualisation inspired by and depending on Gviz. It introduces new types of track and extends the existing ones in order to deal with amino-acid based data such as peptide microarray.

GitHub

HIV.db

HIV.db is a collection of annotations for HIV (hxb2) and SIV (mac239) sequences. It includes genes, gene products, genomic structure elements as well as known epitopes. The selected elements can be plotted against the data using Pviz
